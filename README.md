## Main Functions
1. Score data in calendar
2. Score changes in realtime
3. Stats of score which can give user feedback

- Gaze tracking을 통해 비대면 강의 시 집중도 파악
- 실시간으로 점수가 변하며, 최종 점수는 다이어리에 자동으로 기록
- 집중하지 못 하는 경우 실시간 피드백 진행
